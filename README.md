##Setup

1.Launch mongdb

```
mongod
```
2.Install 

```
cd LG-Blog

npm install
```
3.Start

```
npm start
```
open browser and input:

####http://localhost:3001/

Login:

####http://localhost:3001/login


Development:

```
npm run dev
```
